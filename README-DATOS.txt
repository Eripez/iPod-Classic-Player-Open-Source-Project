========================================
      GESTIÓN DE DATOS - oPud Classic
========================================

El programa guarda automáticamente:
------------------------------
✓ Canciones favoritas
✓ Playlists personalizadas
✓ Estadísticas de reproducción
✓ Configuración del programa
✓ Tema seleccionado
✓ Carpeta de música

UBICACIÓN DE LOS DATOS:
------------------------------
Windows: C:\Users\[TU_USUARIO]\AppData\Roaming\ipod-classic-player\
macOS: ~/Library/Application Support/ipod-classic-player/
Linux: ~/.config/ipod-classic-player/

ARCHIVOS GUARDADOS:
------------------------------
- settings.json (configuración general)
- localStorage (favoritos, playlists, estadísticas)

RESPALDO MANUAL:
------------------------------
Para respaldar tus datos:
1. Copia la carpeta completa de la ubicación arriba indicada
2. O exporta desde el programa (próximamente)

RESTAURAR DATOS:
------------------------------
1. Cierra el programa
2. Copia tus archivos de respaldo a la carpeta de datos
3. Reinicia el programa

NOTA: Los datos se guardan automáticamente cada vez que:
- Añades o quitas favoritos
- Creas o modificas playlists
- Cambias la configuración
- Cierras el programa
